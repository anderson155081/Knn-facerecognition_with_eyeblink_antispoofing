# Senior-Project


## Features

We use the python face-recognition libary to do the KNN-facerecognition, the program include eye-blink for anti-spoofing.

## Requirement:
```
opencv
python3
numpy
imutils
sklearn
pickle
face_recognition
```

### Register new data:
```bash
$ python register.py
```

### Train the data:

```bash
$ python train.py
```

### To run the Program:

```bash
$python main.py
```
